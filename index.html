Programming Languages:

Front-End Development:

HTML 5

HTML is used for structuring content on the web. It's the backbone of web design, ensuring content is correctly presented in the browser.

CSS 3

CSS styles HTML elements, making web pages attractive. CSS3 introduces animations, gradients, and flexible layouts.

JavaScript

JavaScript brings interactivity to websites. It allows dynamic content updates, multimedia control, and animation, transforming static pages into interactive experiences.

Back-End Development:

PHP

PHP is a server-side scripting language used to create dynamic and interactive web pages. It is a popular choice for web development due to its ease of use and robust features.

Java (Basics)

Java is an object-oriented programming language used for building robust, secure, and scalable applications. It is widely used in enterprise applications and Android development.

Front-End Libraries & Frameworks:

React

React is a powerful JavaScript library commonly used for building interactive user interfaces. It enables the development of efficient, fast, dynamic, and reusable UI components

Bootstrap

Bootstrap is a popular open-source framework used for building responsive and mobile-first websites. It provides pre-built components and templates for faster web development.

jQuery

jQuery simplifies JavaScript tasks such as DOM manipulation and event handling. It helps developers quickly build interactive websites with less code.

Node.js

Node.js is open-source runtime environment runs JavaScript on the server side. It's for building scalable network applications, especially APIs and web servers.

Database Management Systems (DBMS):

MySQL

MySQL is an open-source relational database management system (RDBMS) used for storing structured data. It is known for its reliability and scalability in web applications.

MongoDB

MongoDB is a NoSQL database that stores data in flexible, JSON-like documents. It is highly scalable, works well with applications require handling large volumes of data.

Version Control & Collaboration:

Git

Git is a distributed version control system that tracks changes your codebase. It allows multiple developers to collaborate and merge changes without conflicting with each other.

GitHub

GitHub is a platform for hosting and sharing code by many repositories . It integrates with Git to provide version control, collaborative features, and project management tools.

Deployment & Hosting:

Netlify

Netlify is a platform for continuous deployment of web projects. It automates the build and deployment process, making it easy to deploy websites directly from Git repositories.

Vercel

Vercel is a cloud platform for static sites and serverless functions. It allows rapid deployment of web applications and integrates seamlessly with Next.js and other front-end frameworks.
